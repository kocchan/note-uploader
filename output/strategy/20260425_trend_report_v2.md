## ⚡ コンテンツフォーマット方針（重要・更新）

### 「プロンプト集」からの脱却

これまでの記事はプロンプト集・手順解説が中心だったが、競合増加により差別化が困難になっている。

**新方針：失敗談・アンチパターン型**

| 旧フォーマット（脱却） | 新フォーマット（採用） |
|----------------------|----------------------|
| 「使えるプロンプト〇選」 | 「やってみたら全部ダメだった話」 |
| 「〇〇する方法」 | 「〇〇しようとして気づいたこと」 |
| ハウツー・手順解説 | 体験記・試行錯誤のリアル |
| 再現性重視 | 共感・納得感重視 |
| 誰でも書ける | 本人にしか書けない |

**なぜ失敗談が刺さるか**
- ADHDは「うまくいった話」より「失敗した話」に強く共鳴する
- 「自分だけじゃなかった」という安心感が購買動機になる
- プロンプトのコピーは誰でもできるが、失敗の文脈は唯一無二
- 「でもこれだけは効いた」という1点突破が記憶に残る

**記事の基本構造（失敗談型）**
```
① 共感フック：「こういう悩みでAIを試した」
② 失敗の連鎖：「やってみたら〇〇がダメだった」（3〜5パターン）
③ 失敗の分析：「なぜ効かなかったか、ADHD脳の話」
④ 唯一の突破口：「でもこれだけは続いた／効いた」
⑤ 読者へのバトン：「あなたはどの失敗に共感した？」
```

## 新規調査トレンド（v2追加分）

### 1. Claudeにキャラ設定→ADHD専属エージェント化
- **需要**: 高（急上昇中）
- **鮮度**: 今週話題（2026年4月）
- **ADHD向け切り口**: 「ADHDの私が、Claudeに『キャラ』を与えたら仕事の先延ばしが減った話」という実体験noteが話題。Claudeを「厳しめのコーチ」「優しい相棒」などキャラ設定することで、ADHDが苦手な継続・習慣化を支援できる
- **差別化ポイント**: ChatGPTよりClaudeで語られ始めている新切り口。既公開の「脳内整理術」とは別角度
- **参考URL**: [ADHDの私が、Claudeに「キャラ」を与えたら仕事の先延ばしが減った話 - note](https://note.com/pichikyo/n/nb26ee84ebb8e)

### 2. ADHD×感情調節×AIの壁打ち術
- **需要**: 中〜高
- **鮮度**: やや新しい（ニッチだが需要確実）
- **ADHD向け切り口**: ADHDは感情調節困難（感情爆発・自己嫌悪・衝動的な言動）を抱えやすい。AIを「感情の壁打ち相手」として使うことで、人に言えない感情を整理し、衝動的な行動を抑制できる
- **差別化ポイント**: 仕事術ではなくメンタル面のAI活用。競合が少なく、深刻な悩みなので有料記事として購入されやすい
- **参考URL**: [AIでメンタルケアの常識が変わる - PR TIMES](https://prtimes.jp/main/html/rd/p/000000007.000141022.html)

### 3. ChatGPTからClaudeへの乗り換え×ADHD
- **需要**: 高
- **鮮度**: 今週話題
- **ADHD向け切り口**: 「ChatGPTから乗り換え急増中。『Claude』を専属エージェントにして仕事を自動化するコツ3選」がバズ中。ADHDの人が「なぜClaudeの方が使いやすいか」という観点で書けば刺さる（長文理解力・指示の柔軟さ・記憶力）
- **参考URL**: [ChatGPTから乗り換え急増中。「Claude」を専属エージェントにして仕事を自動化するコツ3選 - Lifehacker](https://www.lifehacker.jp/article/2604-matome-chatgpt-claude/)

### 4. ADHD×仕事のミス・忘れ物×AI防止策
- **需要**: 高（継続的）
- **鮮度**: 継続的に人気
- **ADHD向け切り口**: 「仕事でミスばかり」「忘れ物が多い」はADHDの最大の悩みの一つ。AIを使ったダブルチェック・リマインド・確認フロー設計など、具体的なプロンプトと組み合わせた「ミス防止システム」として記事化できる
- **参考URL**: [ADHDの仕事ミス対策ガイド｜特性を強みに変える環境調整と最新ツール活用法](https://di-agent.dandi.co.jp/tips/entry253)

### 5. AIボディダブル×ADHD（既報告からの深掘り）
- **需要**: 高
- **鮮度**: 継続的に人気（競合あり）
- **ADHD向け切り口**: 「誰かがそばにいると集中できる」ADHD特有の「ボディダブル効果」をAIで再現する方法。一人暮らし・在宅勤務ADHDに特に刺さる。作業開始の儀式・声かけプロンプト・進捗報告フローを具体的に提供

## 次回記事テーマ推奨（失敗談フォーマット・優先度順）

### 🥇 推奨度A：AIでタスク管理を試みて、余計カオスになった話

**コア悩み**: タスク管理・優先順位・先延ばし  
**失敗談の種**: 「ChatGPTにタスクを入力しても、結局どれも手をつけなかった」

**タイトル案**:
- 「AIに全タスクを渡したら、やることが増えて詰んだ話【ADHD失敗録】」
- 「タスク管理AIを3つ試して全部挫折した私が、唯一続いた方法」
- 「『AIで先延ばし解消』を信じた3ヶ月と、その後に気づいたこと」

**記事の骨格案**:
- 失敗①：ToDoをAIに作らせたら長すぎて見るのが嫌になった
- 失敗②：優先順位を聞くたびに答えが変わって混乱した
- 失敗③：AIが丁寧すぎて、返答を読むのが億劫になった
- 唯一効いたこと：「今日やること1個だけ言って」という問いかけ

**強み**: 先延ばし攻略の続編として読める・「あるある」の共感度が高い

---

### 🥈 推奨度B：感情が爆発する前にAIに相談して、逆効果だった話

**コア悩み**: 感情調節困難・衝動的な言動・自己嫌悪  
**失敗談の種**: 「AIに怒りをぶつけたら、冷静な返答に余計イライラした」

**タイトル案**:
- 「感情をAIにぶつけたら、正論を返されてもっと病んだ話」
- 「AIを感情の壁打ち相手にしようとして、3つ失敗した。でも4つ目で変わった」
- 「ADHD×感情爆発。AIに救われたのではなく、AIで気づいたこと」

**記事の骨格案**:
- 失敗①：感情を吐き出したら「それは辛かったですね」で終わった
- 失敗②：解決策を提案されても「そんなこと分かってる」ってなった
- 失敗③：AIが正しすぎて、自己嫌悪が深まった
- 唯一効いたこと：感情ではなく「事実だけを」箇条書きで入力する方法

**強み**: ブルーオーシャン・メンタル系で拡散力が高い・深刻な悩みで購買意欲UP

---

### 🥉 推奨度C：Claudeに「優しいコーチ」を演じさせたら、甘やかされて終わった話

**コア悩み**: 習慣化・継続・自己管理  
**失敗談の種**: 「AIキャラ設定をしたが、コーチが優しすぎてサボり続けた」

**タイトル案**:
- 「AIを厳しいコーチにしたら優しく諭されて、甘いコーチにしたら褒められて終わった話」
- 「Claudeに3パターンのキャラを演じさせて、全部ダメだった。でも4つ目で変わった」
- 「AIコーチ設定を試した1ヶ月の失敗と、ADHD脳に本当に刺さった問いかけ」

**記事の骨格案**:
- 失敗①：厳しいキャラ → プレッシャーで開くのが嫌になった
- 失敗②：優しいキャラ → 褒めてくれるから満足して何もしなかった
- 失敗③：キャラなし → AIっぽい返答で感情移入できなかった
- 唯一効いたこと：キャラではなく「問いかけの型」を固定する方法

**強み**: 2026年Claudeブームに乗れる・失敗→発見の流れが明快
