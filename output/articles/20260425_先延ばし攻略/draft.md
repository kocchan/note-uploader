## なぜChatGPTがADHDの先延ばしに効くのか

少しだけ、脳の話をさせてください。

ADHDの脳が「先延ばし」をしてしまう理由は、主に3つあります。

**1. タスクがデカすぎて、どこから手をつけていいかわからない**

「レポートを書く」って言われても、「何から始めればいいの？」ってなりませんか？

普通の人は無意識にタスクを分解できるんです。
でも、ADHDの脳はこれが苦手なんですよね。

**2. 「最初の一歩」を踏み出すエネルギーがない**

ADHDの脳はドーパミンが少ないから、「よし、やるか」という起動エネルギーが出にくい。
だから、やる気はあるのに体が動かない。

こういうの、あるよね？

**3. 「まだ時間ある」と思ってしまう**

ADHDは時間感覚がズレやすい。
1週間後の締め切りが「遠い未来」に感じて、「今日やらなくても大丈夫」と思ってしまう。

そして、ChatGPTはこの3つを全部カバーしてくれるんです。

- タスク分解 → AIがやってくれる
- 最初の一歩 → AIが「これだけやって」と教えてくれる
- 時間感覚 → AIが逆算して「今日やるべきこと」を出してくれる

つまり、**ADHDの脳が苦手なことを、AIに丸投げできる**ってこと。

ここから先は、具体的なプロンプトを紹介していきます。

## 実践プロンプト① 朝の「動き出しスイッチ」

### なぜこれが効くのか

朝起きて、スマホを見る。SNSを開く。気づいたら1時間経ってる。

こういうの、あるよね。

これが起こるのは、「今日何やるか」が明確じゃないから。
脳が「何すればいいかわからない」状態だと、刺激を求めてSNSに逃げてしまう。

このプロンプトを使うと、**朝起きた瞬間に「今日の最重要タスク」がわかる**。それだけ。

脳に「これやればいいんだ」と教えてあげることで、動き出しやすくなります。

### 使い方

1. 朝起きたら、まずChatGPTを開く（SNSより先に）
2. 以下のプロンプトをコピペ
3. 「今日やるべきこと」を入力（箇条書きでOK）

### プロンプト

```
あなたは私の「朝の起動アシスタント」です。

私はADHDの傾向があり、朝の動き出しが苦手です。
以下のタスクリストから、今日の「最重要タスク1つ」を選んでください。

選ぶ基準：
- 締め切りが近いもの
- やらないと他の人に迷惑がかかるもの
- 15分以内に「着手」できるもの

そして、その最重要タスクの「最初の5分でやること」を具体的に教えてください。

---
【今日のタスクリスト】
・[ここにタスクを入力]
・[ここにタスクを入力]
・[ここにタスクを入力]
```

### 実行例

**入力:**
```
【今日のタスクリスト】
・企画書の作成（金曜締め切り）
・経費精算（月末まで）
・チームミーティングの資料準備（明日14時）
・メール返信5件
```

**出力:**
```
🎯 今日の最重要タスク：チームミーティングの資料準備

理由：明日14時が締め切りで最も近く、準備不足だとチームに迷惑がかかります。

📌 最初の5分でやること：
1. 新しいスライドファイルを作成する
2. タイトルと日付だけ入力する

これだけでOKです。まずはファイルを開くところから始めましょう！

- 「最初の5分」が明確になると、動き出しのハードルが一気に下がる

## 実践プロンプト② タスク分解マシン

### なぜこれが効くのか

「レポートを書く」

このタスクを見た瞬間、脳がフリーズしませんか？

ADHDの脳は、**抽象的なタスクを具体的なステップに分解するのがとても苦手**なんです。

だから「何から始めればいいかわからない」→「やる気が出ない」→「先延ばし」のループにハマってしまう。

このプロンプトを使うと、**デカいタスクが「15分でできるサイズ」に砕かれる**。

脳が「これならできそう」と思えるサイズになれば、着手のハードルが下がります。

### 使い方

1. 「やらなきゃいけないけど、手がつかないタスク」を思い浮かべる
2. 以下のプロンプトをコピペ
3. タスク名を入力

### プロンプト

```
あなたは「タスク分解の専門家」です。

私はADHDの傾向があり、大きなタスクを前にするとフリーズしてしまいます。

以下のタスクを「15分以内で完了できるサイズ」に分解してください。

分解のルール：
- 各ステップは具体的な「行動」で書く（「考える」ではなく「〇〇を書き出す」など）
- 最初のステップは「5分以内」でできるものにする
- 全体で10ステップ以内に収める

---
【分解してほしいタスク】
[ここにタスクを入力]
```

### 実行例

**入力:**
```
【分解してほしいタスク】
来週のプレゼン資料を作成する
```

**出力:**
```
📋 タスク分解結果：「来週のプレゼン資料を作成する」

【ステップ1】5分
→ 新しいPowerPointファイルを作成し、タイトルスライドだけ作る

【ステップ2】10分
→ 伝えたいことを3つ、メモ帳に箇条書きで書き出す

【ステップ3】15分
→ 3つのポイントをそれぞれ1スライドにする（文字だけでOK）

【ステップ4】15分
→ 各スライドに補足説明を2-3行追加する

【ステップ5】10分
→ 必要なデータ・グラフがあれば、どこから取るかメモする

【ステップ6】15分
→ データ・グラフを挿入する

【ステップ7】10分
→ 全体を通して読み、流れを確認する

【ステップ8】10分
→ デザインを整える（フォント、色、配置）

【ステップ9】5分
→ ファイルを保存し、共有フォルダにアップする

💡 まずはステップ1だけやってみてください。5分で終わります！
```

### 僕が使ってみた結果

これは、僕が一番お世話になっているプロンプトかもしれません。

以前、会社で「3日後までに報告書を出して」と言われたとき、頭が真っ白になりました。「報告書を書く」というタスクがデカすぎて、何から手をつければいいかわからない。結局2日間何もできず、最終日に徹夜するパターン。

でも、このプロンプトを使ったら、「まず過去の報告書を開いて、構成を確認する（5分）」というステップが出てきたんです。

「それだけでいいの？」と思いながらやってみたら、不思議と次のステップにも手が伸びました。気づいたら2時間で8割終わってた。

あのときの「え、もうこんなに進んでる」という感覚は、今でも覚えています。

### ポイント

- **「まず最初のステップだけ」やればOK**という気持ちで使ってみて
- 分解されたタスクを見て「多いな...」と思ったら、さらに分解を依頼してもOK
- 「考える」系のタスクも「〇〇を書き出す」に変換すると着手しやすい

## 実践プロンプト③ 締め切り逆算スケジューラー

「締め切りは来週の金曜だから、まだ余裕あるな」

これ、ADHDの最大のトラップ。

僕たちの脳は、**時間感覚がズレやすい**。
1週間後が「遠い未来」に感じて、「今日やらなくても大丈夫」と思ってしまう。

気づいたら前日になっている。よくあるやつ。

このプロンプトを使うと、**締め切りから逆算して「今日やるべきこと」が具体的に出てくる**。

「まだ時間ある」の罠から抜け出せます。

### 使い方

1. 締め切りがあるタスクを思い浮かべる
2. 以下のプロンプトをコピペ
3. タスク名と締め切り日を入力

### プロンプト

```
あなたは「締め切り逆算の専門家」です。

私はADHDの傾向があり、「まだ時間ある」と思って先延ばしにしがちです。

以下のタスクを締め切りから逆算して、「今日やるべきこと」を明確にしてください。

ルール：
- 今日を含めた日別のスケジュールを作る
- 各日のタスクは「1-2時間で終わるサイズ」にする
- 締め切り前日は「予備日」として空けておく
- 今日やることは特に具体的に書く

---
【タスク】[ここに入力]
【締め切り】[ここに入力]
【今日の日付】[ここに入力]
```

### 実行例

**入力:**
```
【タスク】四半期レポートの作成
【締め切り】4月30日（水）
【今日の日付】4月25日（金）

**出力:**

→ 各セクションの本文を書く（箇条書きでOK）
⏰ 目安：2時間

【4月28日（月）】
→ グラフ・図表を作成し、挿入する
→ 全体の流れを確認する
⏰ 目安：2時間

【4月29日（火）予備日】
→ 修正・調整用。順調なら休んでOK

【4月30日（水）締め切り】
→ 最終確認して提出

2. 見出しを5つ書く

まずはドキュメントを開くところから！

### 僕が使ってみた結果

これを使い始めてから、締め切り前日の徹夜がほぼなくなりました。

以前は、1週間後の締め切りを見ても「まだ6日もある」と思ってしまい、結局ギリギリまで動けなかった。

でも、このプロンプトを使うと「今日やること」が具体的に出てくるので、「まだ時間ある」と思う暇がないんです。

先月、2週間後が締め切りのプロジェクトがあったんですが、このプロンプトで逆算したら「今日は〇〇だけやればいい」というのが毎日明確になった。結果、締め切りの3日前には終わっていました。

「余裕を持って終わる」という経験、久しぶりすぎて逆に落ち着かなかったくらいです。

### ポイント

- **予備日を必ず入れる**。ADHDは想定外のことが起きやすいので
- 「今日やること」が明確になったら、すぐ着手してみて
- 予定通りいかなくても自分を責めない。予備日があるから大丈夫

## 実践プロンプト⑤ 「やりたくない」言語化セラピー

### なぜこれが効くのか

「やらなきゃいけないのはわかってる。でも、やりたくない」

この感情、よくわかります。

ADHDの脳は、**「やりたくない」という感情に支配されやすい**んです。

そして、その感情を言語化できないから、モヤモヤしたまま動けない。

これは、**「やりたくない」を言語化して、感情を整理する**ためのプロンプト。セラピーみたいな感じ。

なぜやりたくないのかがわかると、意外と「じゃあやるか」ってなれることがあります。

### 使い方

1. 「やりたくないタスク」を思い浮かべる
2. 以下のプロンプトをコピペ
3. 正直に「やりたくない理由」を吐き出す

### プロンプト

```
あなたは私の「感情整理カウンセラー」です。

私はADHDの傾向があり、「やりたくない」という感情に支配されて動けなくなることがあります。

今、以下のタスクをやりたくないと感じています。

【やりたくないタスク】[ここに入力]

これから、私が「やりたくない理由」を正直に吐き出します。
否定せず、共感しながら聞いてください。
そして、最後に「じゃあどうする？」を一緒に考えてください。

では、始めます。
```

### 実行例

**入力:**
```
【やりたくないタスク】上司へのメール返信
```

**あなたの吐き出し:**
```
やりたくない。
なんか、上司のメールの言い方がイラッとする。
あと、返信の内容を考えるのがめんどくさい。
どう書いても「これでいいのかな」って不安になるし。
そもそもメール嫌い。
```

**ChatGPTの返答:**
```
わかります。上司のメールって、なんかイラッとすること多いですよね。

整理すると、やりたくない理由は3つありそうです：
1. 上司の言い方にモヤモヤしてる
2. 返信の内容を考えるのがエネルギー使う
3. 「これでいいのか」という不安

どれが一番大きいですか？

...

なるほど、「これでいいのか」の不安が一番なんですね。

じゃあ、こうしてみませんか？
「返信のドラフトを私（ChatGPT）に書かせる」→「それを修正する」

ゼロから考えるより、たたき台があったほうがラクじゃないですか？
上司のメールをコピペしてくれたら、返信案を作りますよ。
```

### 僕が使ってみた結果

経費精算がどうしてもできなくて、3ヶ月も溜め込んでいたことがあります。

「やらなきゃ」と思うたびに、なぜか体が動かない。自分でも理由がわからなかった。

このプロンプトで「なぜやりたくないか」を吐き出したら、「領収書をなくしてるかもしれない」という不安が一番大きいことに気づいたんです。

ChatGPTに「じゃあ、まずは領収書があるか確認するだけやってみたら？」と言われて、それだけやってみた。全部あった。

不安が消えたら、なぜかスルスルと経費精算が進みました。「やりたくない」の正体を知るだけで、こんなに違うのかと驚きました。

### ポイント

- **正直に吐き出す**のが大事。カッコつけなくていい
- 「やりたくない理由」がわかると、解決策が見えてくる
- このプロンプト自体が「着手のきっかけ」になることも多い

## 実践プロンプト⑥ 5分だけスタート誘導

### なぜこれが効くのか

「やる気が出たらやろう」

これ、永遠に来ないんですよね。

ADHDの脳は、**やり始めないとやる気が出ない**。

逆に言うと、5分でもやり始めれば、脳がエンジンかかって続けられることが多いんです。

このプロンプトは、**「5分だけやってみて」と促して、5分後に「続ける？やめる？」を聞いてくる**。

過集中モードへの入口を作ります。

### 使い方

1. 「絶対やりたくないタスク」を思い浮かべる
2. 以下のプロンプトをコピペ
3. 5分だけやってみる
4. 5分後にChatGPTに報告

### プロンプト

```
あなたは私の「5分スタートコーチ」です。

私はADHDの傾向があり、着手のハードルが高いです。
でも、5分だけでもやり始めれば、続けられることが多いです。

今から以下のタスクを「5分だけ」やります。

【タスク】[ここに入力]

5分後に報告します。
5分やったら、「続ける？やめる？」を聞いてください。

続けると言ったら、励ましてください。
やめると言っても、責めないでください。5分やっただけでも偉いので。

では、5分タイマースタートします。
```

### 実行例

**入力:**
```
【タスク】確定申告の書類整理
```

**5分後の報告:**
```
あなた：5分やりました。領収書を10枚くらい仕分けした。
```

**ChatGPTの返答:**
```
5分で領収書10枚！十分すぎます。

さて、どうしますか？

A) もう5分だけ続ける
B) 今日はここまでにする

どっちでも正解ですよ。5分やった時点でもう勝ちですから。
```

**続ける場合:**
```
あなた：もう5分だけやる
```

**ChatGPTの返答:**
```
いいですね！その調子。

次の5分で、あと10枚仕分けしましょう。
タイマースタート！

終わったらまた報告してくださいね。
```

### 僕が使ってみた結果

確定申告の書類整理、毎年地獄でした。

「今年こそ早めにやろう」と思っても、結局3月に入ってから泣きながら徹夜するパターン。

今年は、このプロンプトを使って「5分だけ」を繰り返しました。

「5分だけ領収書を仕分ける」「5分だけ入力する」を毎日続けたら、なんと2月中に終わったんです。しかも徹夜なし。

不思議なのは、「5分だけ」と言いながら、気づいたら30分くらいやってることが多かったこと。脳のエンジンがかかると、止まらなくなるみたいです。

### ポイント

- **「5分だけ」が魔法の言葉**。脳が「それくらいならいいか」と思える
- 5分やって「やめる」を選んでも、それは失敗じゃない。5分やった自分を褒めよう
- 意外と「もうちょっとやるか」ってなることが多い

## 実践プロンプト⑦ 1日の振り返り＆明日の準備

### なぜこれが効くのか

ADHDの脳は、**「今」しか見えない**。

だから、今日何ができたかを振り返らないし、明日の準備もしない。

そして翌朝「今日何やるんだっけ？」からスタートして、またSNSに逃げる。

こういうパターン、あるよね。

このプロンプトを使うと、**夜に今日の振り返りをして、明日の「最重要タスク1つ」を決められる**。

翌朝の「動き出しスイッチ」の準備になります。

### 使い方

1. 寝る前（または仕事終わり）にChatGPTを開く
2. 以下のプロンプトをコピペ
3. 今日やったことを報告

### プロンプト

```
あなたは私の「1日の振り返りパートナー」です。

私はADHDの傾向があり、今日できたことを忘れがちです。
また、明日の準備をせずに寝てしまい、翌朝迷子になります。

今日の振り返りと、明日の準備を手伝ってください。

【今日やったこと】
[ここに入力（箇条書きでOK）]

【明日やらなきゃいけないこと】
[ここに入力（箇条書きでOK）]

以下をお願いします：
1. 今日できたことを褒めてください（小さなことでも）
2. 明日の「最重要タスク1つ」を選んでください
3. 明日の朝、最初にやる「5分アクション」を決めてください
```

### 実行例

**入力:**
```
【今日やったこと】
・メール5件返信した
・企画書の見出しだけ作った
・会議に出た
・経費精算を途中までやった

【明日やらなきゃいけないこと】
・企画書の続き
・経費精算の続き
・クライアントに電話
・チームミーティングの準備
```

**ChatGPTの返答:**
```
📝 今日の振り返り

【できたこと】
- メール5件返信 → 地味だけど、これが一番大変なやつ。えらい！
- 企画書の見出し作成 → ゼロから1を作った。最初の一歩を踏み出しましたね！
- 会議に出た → 当たり前に見えて、ADHDには消耗するやつ。お疲れさまでした。
- 経費精算を途中まで → 「途中まで」でも手をつけた自分を褒めましょう。

今日も1日お疲れさまでした。十分がんばりましたよ。

---

📌 明日の最重要タスク：企画書の続き

理由：見出しまでできてるから、勢いがあるうちに進めたいですね。

🌅 明日の朝、最初の5分でやること：
「企画書を開いて、最初の見出しの中身を箇条書きで3つ書く」

これだけです。5分で終わります。
朝起きたら、SNS開く前にこれだけやってみてください。

おやすみなさい。
```

### 僕が使ってみた結果

寝る前にこれをやるようになってから、翌朝のスタートが全然違います。

以前は、朝起きて「今日何やるんだっけ...えーと...」と考えるところから始まっていました。これがストレスで、結局SNSに逃げる。

でも今は、前の夜に「明日の朝、最初にやること」が決まっているから、起きたらそれをやるだけ。考える必要がない。

あと、「今日できたことを褒めてもらう」のが意外と効きました。

「メール5件返信した」なんて当たり前のことでも、「それ、地味に大変なやつですよね。えらい！」と言われると、なんか救われる。自分ではなかなか自分を褒められないので、AIに褒めてもらうのはアリだなと思っています。

### ポイント

- **今日できたことを「褒めてもらう」**のが大事。ADHDは自己肯定感が低くなりがち
- 明日の「5分アクション」を決めておくと、翌朝の起動がラクになる
- 寝る前のルーティンにすると効果的

## 【応用】ChatGPTのMemory機能で「自分専用AI秘書」を作る

ここまで7つのプロンプトを紹介しましたが、毎回「私はADHDの傾向があり...」と書くのは面倒ですよね。

そこで、**ChatGPTのMemory機能**を使います。

Memory機能は、ChatGPTがあなたのことを「覚えてくれる」機能。
一度設定すれば、次回から自動的にあなたの特性を考慮した回答をしてくれます。

### 設定方法

1. ChatGPTの設定画面を開く
2. 「Personalization」→「Memory」をオンにする
3. 以下を伝えて、覚えさせる

```
以下のことを覚えておいてください。

【私の特性】
- ADHDの傾向がある
- 朝の動き出しが苦手
- 大きなタスクを前にするとフリーズする
- 先延ばし癖がある
- 時間感覚がズレやすい
- 作業中に脱線しやすい

【私への接し方】
- 責めないでください
- 小さな成功も褒めてください
- タスクは「15分以内でできるサイズ」に分解してください
- 「最初の5分でやること」を具体的に教えてください
- 共感的な言葉をかけてください

【効果的だった声かけ】
- 「5分だけやってみよう」
- 「これだけでOK」
- 「十分がんばったよ」
```

### 設定後の使い方

一度覚えさせれば、次回からは：

```
あなた：今日のタスク整理して

ChatGPT：（あなたがADHDで、先延ばし癖があることを踏まえて）
了解！まず今日やらなきゃいけないこと、教えて。
優先順位つけて、最初の5分でやることを一緒に決めよう。
```

こんな感じで、**毎回説明しなくても、あなたに合った対応をしてくれます**。
