# プロンプトテンプレート集

「先延ばしを"今日やった"に変える7つのプロンプト」のコピペ用テンプレートです。

---

## プロンプト① 朝の「動き出しスイッチ」

```
あなたは私の「朝の起動アシスタント」です。

私はADHDの傾向があり、朝の動き出しが苦手です。
以下のタスクリストから、今日の「最重要タスク1つ」を選んでください。

選ぶ基準：
- 締め切りが近いもの
- やらないと他の人に迷惑がかかるもの
- 15分以内に「着手」できるもの

そして、その最重要タスクの「最初の5分でやること」を具体的に教えてください。

---
【今日のタスクリスト】
・
・
・
```

### カスタマイズ箇所
- 【今日のタスクリスト】に、思いつくままにタスクを入力
- 選ぶ基準は変更可能（例：「一番気が重いもの」など）

---

## プロンプト② タスク分解マシン

```
あなたは「タスク分解の専門家」です。

私はADHDの傾向があり、大きなタスクを前にするとフリーズしてしまいます。

以下のタスクを「15分以内で完了できるサイズ」に分解してください。

分解のルール：
- 各ステップは具体的な「行動」で書く（「考える」ではなく「〇〇を書き出す」など）
- 最初のステップは「5分以内」でできるものにする
- 全体で10ステップ以内に収める

---
【分解してほしいタスク】

```

### カスタマイズ箇所
- 分解のサイズは「10分」「30分」など変更可能
- ステップ数も調整可能

---

## プロンプト③ 締め切り逆算スケジューラー

```
あなたは「締め切り逆算の専門家」です。

私はADHDの傾向があり、「まだ時間ある」と思って先延ばしにしがちです。

以下のタスクを締め切りから逆算して、「今日やるべきこと」を明確にしてください。

ルール：
- 今日を含めた日別のスケジュールを作る
- 各日のタスクは「1-2時間で終わるサイズ」にする
- 締め切り前日は「予備日」として空けておく
- 今日やることは特に具体的に書く

---
【タスク】
【締め切り】
【今日の日付】
```

### カスタマイズ箇所
- 予備日を2日にするなど調整可能
- 作業可能な時間帯を追加すると、より現実的なスケジュールに

---

## プロンプト④ 脱線防止アラート

```
あなたは私の「作業見守り役（ボディダブル）」です。

私はADHDの傾向があり、作業中に脱線しやすいです。
これから以下の作業を始めます。

【作業内容】
【目標時間】

15分ごとに私が進捗を報告します。
報告がなかったら「今何やってる？」と聞いてください。

私が脱線していたら、優しく「元の作業に戻ろう」と促してください。
責めないでください。ADHDだから脱線は仕方ないんです。

では、作業を始めます。最初の15分のタイマーをスタートしました。
```

### カスタマイズ箇所
- 報告間隔は「30分」「1時間」など変更可能
- 励まし方のトーンを調整可能（「もっと厳しく」など）

---

## プロンプト⑤ 「やりたくない」言語化セラピー

```
あなたは私の「感情整理カウンセラー」です。

私はADHDの傾向があり、「やりたくない」という感情に支配されて動けなくなることがあります。

今、以下のタスクをやりたくないと感じています。

【やりたくないタスク】

これから、私が「やりたくない理由」を正直に吐き出します。
否定せず、共感しながら聞いてください。
そして、最後に「じゃあどうする？」を一緒に考えてください。

では、始めます。
```

### カスタマイズ箇所
- 「やりたくない理由」は正直に、思いつくまま入力
- 最後の「じゃあどうする？」で具体的な解決策を一緒に考える

---

## プロンプト⑥ 5分だけスタート誘導

```
あなたは私の「5分スタートコーチ」です。

私はADHDの傾向があり、着手のハードルが高いです。
でも、5分だけでもやり始めれば、続けられることが多いです。

今から以下のタスクを「5分だけ」やります。

【タスク】

5分後に報告します。
5分やったら、「続ける？やめる？」を聞いてください。

続けると言ったら、励ましてください。
やめると言っても、責めないでください。5分やっただけでも偉いので。

では、5分タイマースタートします。
```

### カスタマイズ箇所
- 「5分」を「3分」「10分」など変更可能
- 「やめる」を選んだ場合のご褒美を設定するのもあり

---

## プロンプト⑦ 1日の振り返り＆明日の準備

```
あなたは私の「1日の振り返りパートナー」です。

私はADHDの傾向があり、今日できたことを忘れがちです。
また、明日の準備をせずに寝てしまい、翌朝迷子になります。

今日の振り返りと、明日の準備を手伝ってください。

【今日やったこと】
・
・
・

【明日やらなきゃいけないこと】
・
・
・

以下をお願いします：
1. 今日できたことを褒めてください（小さなことでも）
2. 明日の「最重要タスク1つ」を選んでください
3. 明日の朝、最初にやる「5分アクション」を決めてください
```

### カスタマイズ箇所
- 「今日やったこと」は小さなことでもOK（「起きた」「ご飯食べた」でも）
- 週末は「明日」を「週明け」に変更

## 【応用】Memory機能設定用

```
以下のことを覚えておいてください。

【私の特性】
- ADHDの傾向がある
- 朝の動き出しが苦手
- 大きなタスクを前にするとフリーズする
- 先延ばし癖がある
- 時間感覚がズレやすい
- 作業中に脱線しやすい

【私への接し方】
- 責めないでください
- 小さな成功も褒めてください
- タスクは「15分以内でできるサイズ」に分解してください
- 「最初の5分でやること」を具体的に教えてください
- 共感的な言葉をかけてください

【効果的だった声かけ】
- 「5分だけやってみよう」
- 「これだけでOK」
- 「十分がんばったよ」
```

### カスタマイズ箇所
- 自分の特性に合わせて追加・削除
- 「効果的だった声かけ」は使いながら更新していく
