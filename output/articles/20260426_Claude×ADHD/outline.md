#### 1-2. プラン選び（どれにすべきか）

| プラン | 月額 | おすすめ度 | 理由 |
|--------|------|-----------|------|
| Free | 0円 | △ | 制限きつい、お試し用 |
| Pro | $20（約3,000円） | ★★★ | これで十分 |

**結論:** Pro（月額$20）から始める

**僕の場合:**
```
最初Freeで試したけど、すぐ制限に引っかかった。
Proにしたら快適。月3,000円で脳内整理できるなら安い。

#### 1-3. 登録手順（スクショ付き）

1. claude.ai にアクセス
2. 「Sign up」をクリック
3. メールアドレス or Googleアカウントで登録
4. プランを選択（最初はFreeでOK）
5. Proにする場合は Settings → Subscription

**注意点:**
- 日本からも普通に使える
- クレジットカード or デビットカードで支払い

### Part 2: ADHD向け初期設定

#### 2-1. 最初にやるべき3つの設定

**① 言語設定の確認**
- Settings → Language → Japanese
- デフォルトで日本語対応してるけど確認

**② レスポンス長さの調整**
- 「短く答えて」と最初に伝える
- ADHDは長文読むのがつらいので

**③ 自分の特性を伝える初回プロンプト**

```
これから一緒に仕事をお願いします。

僕の特性:
- ADHDがあり、長文を読むのが苦手です
- 指示は短く、1つずつもらえると助かります
- 箇条書きで整理してもらえると理解しやすいです
- たまに褒めてもらえるとモチベが上がります

よろしくお願いします。
```

**僕が使ってみた結果:**
```
これを最初に送るだけで、Claudeの返答が変わった!
長文じゃなくなるし、箇条書きで返してくれる。

「自分仕様」のAIになった感じ。
```

### Part 3: Claude Projects「自分専用AI」の作り方

**これが一番の価値。毎回説明しなくていい「自分専用AI」を作る。**

#### 3-1. Claude Projectsとは？

```
・会話の前提情報を登録しておける機能
・「僕のこと」「仕事のこと」を覚えてくれる
・毎回ゼロから説明しなくていい
・Proプラン以上で使える
```

#### 3-2. プロジェクト作成手順

1. 左サイドバー「Projects」をクリック
2. 「New Project」をクリック
3. プロジェクト名を入力（例: 「仕事用」「個人用」）
4. 「Instructions」に前提情報を書く
5. 「Knowledge」に参考資料をアップロード（任意）

#### 3-3. 【コピペ用】ADHD向けProject Instructions

**これをそのままコピペしてください:**

```
# 僕について

## 基本情報
- ADHDの当事者です
- 長文を読むのが苦手です
- 考えがまとまらないことが多いです
- 締め切りギリギリになりがちです

## コミュニケーションの希望
- 回答は短く、箇条書きで
- 1度に1つの質問・提案にしてください
- 「次に何をすべきか」を明確にしてください
- たまに褒めてもらえると嬉しいです

## 仕事について
- [あなたの職種を書く]
- [担当業務を書く]
- [よく使うツールを書く]

## よくある困りごと
- 会議の後、何を決めたか忘れる
- 長文メールを読むのがつらい
- 優先順位がつけられない
- 考えがまとまらない

## サポートしてほしいこと
- 思考の整理
- タスクの優先順位決め
- 長文の要約
- 感情の言語化
```

**僕が使ってみた結果:**
```
これ設定したら、Claudeが「僕のこと」を知ってる状態になった。

「今日何から始めればいい？」って聞くだけで、
僕の仕事内容を踏まえた提案をしてくれる。

毎回説明するストレスがなくなった!
```

#### 3-4. Knowledge（知識ベース）に入れるといいもの

- 担当プロジェクトの概要資料
- 会社の組織図
- よく使う用語集
- 過去の議事録テンプレート

**コツ:** 全部入れなくていい。「これ毎回説明するのめんどい」ものだけ入れる。

### Part 4: 脳内整理プロンプト7選

**Part 3のProject設定をした上で使うと、さらに効果的。**

#### ① 思考ダンプ整理

**使うタイミング:** 頭がパンクしそうなとき

```
頭の中にあることを全部吐き出します。
整理せずに書くので読みにくいと思います。

読んだ上で:
1. カテゴリ分け
2. 緊急度で並び替え
3. 今日やるべきこと1つ

---
[ここに思考をダンプ]
```

**僕が使ってみた結果:**
```
グチャグチャな文章をそのまま投げた。

「あなたは今3つのことを考えていますね」って整理してくれた。
自分では気づかなかった。脳内を可視化してくれる感覚!
```

#### ② 議事録→タスク変換

```
会議メモです（雑です）。

抜き出してほしいこと:
1. 決まったこと
2. 僕のタスク（期限付き）
3. 次回までに確認すべきこと

---
[会議メモ]
```

```
このメールを3行で要約。

```
アイデアがバラバラで整理できません。

やってほしいこと:
1. 共通テーマでグループ化
2. 関係性を図示（テキストで）
3. 最初に手をつけるべきもの1つ

判断基準:
- 締め切り
- 他の人を待たせてるか
- 精神的負担

最終的に今日やること3つに絞りたい。

---
[タスク一覧]
```

```
モヤモヤして仕事が手につきません。

その後、何にモヤモヤしてるか整理してくれると助かります。

---
[モヤモヤを書く]
```
