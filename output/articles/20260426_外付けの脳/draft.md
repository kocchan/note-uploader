## ◆ Step 1：自分のADHD特性を棚卸しする

まずは、自分を知るところから。

「いや、自分のことはわかってるよ」って思うかもしれません。

でも、**言語化できていますか？**

AIに伝えるには、言葉にする必要があります。

ここでは2つのワークシートを使います。

---

### ワークシート①：困りごと棚卸しシート

まず、仕事で「困っていること」を全部書き出します。

大事なのは、**具体的に書くこと**。

「集中できない」じゃなくて、「午後3時くらいから集中が切れる」。

「忘れっぽい」じゃなくて、「会議で決まったことを翌日には忘れてる」。

#### 使い方

1. 下のシートをコピーする
2. 最近1週間で「困った」「失敗した」「焦った」場面を思い出す
3. できるだけ具体的に書く
4. 5個以上書けたらOK

#### 記入例

| # | 困りごと | いつ起きる？ | どうなる？ |
|---|---------|------------|----------|
| 1 | タスクを先延ばしにする | 午前中、難しそうなタスクを見たとき | 結局夕方まで手をつけない |
| 2 | 会議の内容を忘れる | 会議の翌日 | 「あれ何だっけ」と聞き直す |
| 3 | メール返信を忘れる | 「後で返そう」と思ったとき | 1週間放置して催促される |
| 4 | 集中が続かない | 午後3時以降 | SNSを見始めて1時間消える |
| 5 | 何から手をつけていいかわからない | タスクが5個以上あるとき | 全部中途半端になる |

### ワークシート②：AIに任せたいこと仕分けシート

困りごとを書き出したら、次は「仕分け」です。

全部をAIに任せる必要はありません。

というか、**任せられないものもあります**。

#### 仕分けの基準

| AIに任せやすい | 自分でやった方がいい |
|--------------|-------------------|
| 情報の整理・要約 | 最終的な判断・決定 |
| 選択肢を出す | 人間関係の対応 |
| リマインド・声かけ | 体を動かすこと |
| 文章の下書き | 感情のケア |
| 手順の言語化 | |

#### 使い方

1. ワークシート①で書いた困りごとを転記
2. 「AIに任せたい部分」を考える
3. 任せ方のイメージを書く

#### 記入例

| 困りごと | AIに任せたい部分 | 任せ方のイメージ |
|---------|----------------|----------------|
| タスクを先延ばしにする | 「最初の一歩」を教えてもらう | 「このタスク、まず何からやればいい？」と聞く |
| 会議の内容を忘れる | 議事録の整理 | メモを投げて要点をまとめてもらう |
| メール返信を忘れる | 返信の下書き | 「このメールに返信して」で下書き作成 |
| 集中が続かない | 作業の区切りを作る | 「25分タイマー + 休憩の声かけ」をお願いする |
| 何から手をつけていいかわからない | 優先順位付け | タスク一覧を投げて「今日やるべき3つ」を聞く |

### 僕の実体験：棚卸しでわかったこと

正直に言うと、最初は「こんなの意味あるの？」と思ってました。

でもやってみたら、発見があったんです。

**僕の困りごとの8割は「始められない」だった。**

「忘れる」「集中できない」と思ってたけど、よく見ると違う。

- タスクを見ても「何からやればいいかわからない」から始められない
- メールを見ても「何て返せばいいかわからない」から後回しにする
- 会議の準備も「どこから手をつければいいかわからない」から逃げる

つまり、僕に必要なのは「忘れない仕組み」じゃなくて「最初の一歩を教えてくれる存在」だった。

これがわかってから、AIの使い方がガラッと変わりました。

## ◆ Step 2：朝・日中・夜のAIルーティンを作る

自分の困りごとがわかったら、次はルーティンを作ります。

ここで大事なのは、**全部やろうとしないこと**。

ADHDの僕らは、「完璧にやろう」として挫折するパターンが多いですよね。

だから、まずは1つだけ。

朝・日中・夜のどれか1つから始めてみてください。

---

### 朝ルーティン：「今日やること」を決める

朝、起きてから仕事を始めるまでの間。

ここで「今日やること」を決められると、1日がだいぶ楽になります。

#### なぜ朝が重要か

ADHDの脳は、選択肢が多いとフリーズします。

朝の時点で「今日はこれをやる」と決めておくと、日中の「何やろう…」が減る。

決断の回数を減らすのがポイントです。

#### 朝ルーティンの流れ

1. 昨日のタスク残りを確認
2. 今日の予定を確認
3. AIに「今日やるべき3つ」を聞く
4. 3つだけ決めて、それ以外は見ない

#### プロンプト例

```
今日のタスク一覧です。

- 企画書の修正
- クライアントへのメール返信
- 来週の会議資料作成
- 経費精算
- 〇〇さんに電話

今日中にやるべきことを3つだけ選んで、理由も教えてください。
僕はADHDで、たくさんあると手がつけられなくなります。
```

#### 僕の朝ルーティン実例

僕は毎朝、コーヒーを入れながらこれをやってます。

所要時間は5分くらい。

最初は「面倒くさい」と思ったけど、これをやらないと午前中がグダグダになることに気づいてから、習慣になりました。

ポイントは、**3つ以上選ばないこと**。

「あれもこれも」と思った瞬間、ADHDの脳はフリーズします。

3つ終わったら、追加してもいい。

でも最初は3つだけ。

### 日中ルーティン：「詰まったら聞く」習慣

日中、仕事をしていて「詰まる」瞬間がありますよね。

- 何から手をつけていいかわからない
- 考えがまとまらない
- やる気が出ない

このとき、**自分で考え込まない**のがコツです。

詰まったら、すぐAIに聞く。

#### ADHDが日中に詰まるパターン

| パターン | 症状 | AIへの聞き方 |
|---------|------|-------------|
| スタートできない | タスクを見ても動けない | 「最初の5分で何をすればいい？」 |
| 考えがまとまらない | 頭の中がごちゃごちゃ | 「今から話すことを整理して」 |
| やる気が出ない | わかってるけど体が動かない | 「このタスクの楽しいポイントを教えて」 |

---

#### 詰まったときのプロンプト3パターン

**パターン1：スタートできないとき**

```
このタスクをやらないといけないんだけど、手がつけられません。

最初の5分で何をすればいいか、具体的に教えてください。
「〇〇を開いて、△△をクリック」くらい具体的にお願いします。
```

**パターン2：考えがまとまらないとき**

```

```

**パターン3：やる気が出ないとき**


### 夜ルーティン：「振り返り」で明日を楽にする

1日の終わりに、5分だけ振り返りをします。

「振り返りなんて面倒」と思いますよね。

僕もそう思ってました。

でもこれ、**明日の自分へのプレゼント**なんです。

#### なぜ振り返りが効くか

ADHDは「昨日の自分」を忘れがち。

「あれ、昨日何してたっけ」から1日が始まると、エンジンがかかるまで時間がかかる。

夜に「明日の自分へのメモ」を残しておくと、翌朝のスタートが楽になります。

#### 振り返りの流れ

1. 今日やったことを書く
2. 明日やることを書く
3. 「明日の自分へ一言」を書く

#### プロンプト例

```
今日の振り返りを手伝ってください。

■ 今日やったこと
- 企画書の修正（完了）
- クライアントへのメール返信（完了）
- 来週の会議資料作成（途中）

■ 明日やること
- 会議資料の続き
- 経費精算

この情報をもとに、「明日の朝の自分」に向けたメモを作ってください。
朝見てすぐ動けるように、具体的にお願いします。
```

### カスタマイズのコツ

ここまで読んで、「全部やるの無理」と思った人。

正解です。

**全部やらなくていいです。**

#### まずは1つだけ

朝・日中・夜のどれか1つだけ選んでください。

おすすめは**朝ルーティン**。

理由は、1日の最初に「今日やること」を決めると、その後が楽になるから。

#### 1週間続けてみる

選んだルーティンを、1週間だけ続けてみてください。

「合わないな」と思ったら変えていい。

大事なのは、**自分に合うものを見つけること**。

他人のやり方をそのまま真似しなくていいんです。

## ◆ Step 3：挫折せずに続ける仕組み

ルーティンを作っても、続かなければ意味がない。

ここがADHDにとって一番難しいところですよね。

僕も何度も挫折しました。

でも、いくつかの仕組みを入れてから、続くようになったんです。

---

### ADHDが挫折するパターン

まず、敵を知りましょう。

| パターン | 原因 | 対策 |
|---------|------|------|
| 飽きる | 新鮮味がなくなる | 定期的に「アップデート」する |
| 忘れる | 習慣化する前に忘れる | トリガーを設定する |
| 面倒になる | ステップが多すぎる | 簡略化する |

---

### 挫折対策チェックリスト

これ、僕が実際に使っているチェックリストです。

1週間に1回、見直しています。

#### チェックリスト（コピペ用）

```
【挫折対策チェックリスト】

■ 飽き対策
[ ] 最近、ルーティンを「アップデート」したか？
[ ] 新しいプロンプトを試したか？
[ ] 「楽しい」と感じる瞬間があるか？

■ 忘れ対策
[ ] ルーティンの「トリガー」は明確か？
    （例：コーヒーを入れたら朝ルーティン）
[ ] リマインダーを設定しているか？
[ ] 目に見える場所にメモがあるか？

■ 面倒対策
[ ] ステップは3つ以内か？
[ ] 5分以内に終わるか？
[ ] テンプレートはコピペですぐ使えるか？

■ 今週の改善点
→
```

---

### 各項目の解説

#### 飽き対策

ADHDは「新しいもの」が好き。

同じことを続けるのが苦手です。

だから、**定期的に「アップデート」する**のがコツ。

プロンプトを少し変えてみる。

テンプレートに項目を追加してみる。

小さな変化を入れることで、新鮮味を保てます。

#### 忘れ対策

習慣化する前に忘れる。

これ、ADHDあるあるですよね。

対策は「トリガー」を設定すること。

僕の場合：
- コーヒーを入れたら → 朝ルーティン
- お昼ご飯の後 → タスク確認
- パソコンを閉じる前 → 夜ルーティン

**既存の習慣にくっつける**のがポイントです。

#### 面倒対策

ステップが多いと、やる気が出ない。

だから、**とにかく簡略化**。

- ステップは3つ以内
- 5分以内に終わる
- コピペですぐ使える

「これくらいならできる」と思えるレベルまで削ぎ落とす。

完璧を目指さない。

---

### 僕が実際にやった挫折対策

正直に言うと、最初の1ヶ月は3回くらい挫折しました。

「やっぱり続かないな」と思いました。

でも、**挫折するたびに仕組みを見直した**んです。

#### Before：3日坊主の常連だった

- 朝のルーティンが15分かかってた → 面倒で続かない
- トリガーを決めてなかった → 気づいたらお昼
- 完璧にやろうとしてた → 1回サボると全部やめる

#### After：90日続いている理由

- 朝のルーティンを5分に短縮
- 「コーヒーを入れたら」をトリガーに
- 「3つのうち1つやればOK」に基準を下げた

特に効いたのは、**「1つやればOK」ルール**。

朝ルーティンの3ステップ、全部やらなくていい。

1つでもやったら、その日は成功。

このハードルの低さが、続く秘訣でした。

## ◆ 90日後の変化

ここまでの方法を90日続けて、何が変わったか。

正直に書きます。

---

### 仕事での変化

#### タスク管理が楽になった

以前は、タスク一覧を見るだけで憂鬱でした。

「どれからやればいいの…」でフリーズ。

今は、朝に「3つだけ」決めるから、迷わない。

**「選ばなくていい」って、こんなに楽なんだ**と思いました。

#### 「何から手をつけていいかわからない」が減った

「詰まったら聞く」を習慣にしてから、止まる時間が減りました。

以前は30分くらい固まってたのが、今は5分で動き出せる。

AIに「最初の5分で何する？」と聞くだけ。

これだけで、全然違います。

#### 会議の準備が怖くなくなった

会議の前日、「何を準備すればいいんだろう」で固まってた。

今は、AIに「この会議の準備、何すればいい？」と聞く。

リストが出てくるから、それを順番にやるだけ。

**「考える」から「聞く」に変えた**のが大きかった。

---

### メンタルの変化

#### 自己嫌悪が減った

「また先延ばしにした」「また忘れた」で自分を責めることが減りました。

AIが「最初の一歩」を教えてくれるから、動けることが増えた。

動けると、自己嫌悪が減る。

このサイクルが回り始めてから、気持ちが楽になりました。

#### 「できない自分」を責めなくなった

これは意外な変化でした。

「AIに任せていい」と思えるようになったら、「全部自分でやらなきゃ」のプレッシャーが減った。

苦手なことは、AIに頼っていい。

**それって、ズルじゃなくて「賢い」んだ**と思えるようになりました。

---

### 正直なところ

完璧じゃないです。

今でもサボる日はある。

先延ばしにすることもある。

でも、**「前よりマシ」が積み重なった**んです。

- 先延ばしの時間が30分 → 10分になった
- 週に1回は徹夜してた → 月に1回くらいに減った
- 毎日自己嫌悪 → 週に1回くらい

劇的な変化じゃないかもしれない。

でも、この「少しずつマシになる」が大事なんだと思います。
