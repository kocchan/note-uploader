# レビュールール

記事のレビュー時にチェックするポイントです。

---

## AIっぽさチェック

| 観点 | NGパターン | OKパターン |
|------|-----------|-----------|
| 書き出し | 「〇〇について解説します」 | 「正直、これで変わった」 |
| 接続詞 | 「また」「さらに」の連続 | 段落を短く、リズムをつける |
| 語尾 | 「〜です」「〜ます」の単調さ | 「〜なんです」「〜だった」を混ぜる |
| 説明 | 網羅的・教科書的 | 「これだけでOK」 |
| 特典 | 最後にちょろっと書く | **冒頭で先に見せる** |

---

## NGワード・表現

- 「〜について解説します」
- 「また」「さらに」「加えて」の連続使用
- 「〜することが重要です」
- 「〜と言えるでしょう」
- 「〜ものです」
- 「様々な」「多くの」（具体性がない）

---

## チェックリスト

### 構成
- [ ] 無料部分が25%以下になっているか
- [ ] 特典が冒頭で明示されているか
- [ ] 有料部分に実用的な価値が5個以上あるか

### 文体
- [ ] 一人称が「僕」「私」になっているか（「俺」はNG）
- [ ] 段落が短いか（1〜3文で改行）
- [ ] 同じ語尾が3回以上連続していないか
- [ ] 読者への問いかけが含まれているか

### 内容
- [ ] CLAUDE.mdのターゲット・テーマに合致しているか
- [ ] 当事者が「自分のことだ」と思える内容か
- [ ] 実体験（Before/After）が含まれているか
- [ ] 価格に見合う価値があるか

### トーン
- [ ] 上から目線になっていないか
- [ ] 押し付けがましくないか
- [ ] 「頑張れ」系の精神論がないか

## 修正の優先度

| 優先度 | 内容 |
|--------|------|
| 高 | AIっぽい表現の修正 |
| 高 | 特典を冒頭に移動 |
| 中 | 語尾のバリエーション追加 |
| 中 | 段落を短く分割 |
| 低 | 言い回しの微調整 |
