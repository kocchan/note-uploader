# 引用ツイートルール

引用ツイートの対象選定と内容作成のルールです。

---

## Part 1: どういうツイートを引用ツイートするか

### ターゲットアカウントの条件

| 優先度 | 条件 |
|--------|------|
| 必須 | CLAUDE.mdのターゲット（ADHD当事者）と関連するテーマで発信している |
| 必須 | フォロワー1,000人以上（リーチ効果が期待できる） |
| 必須 | 直近7日以内に引用していないアカウント（`quote_history.md`で確認） |
| 推奨 | 定期的にエンゲージメントがあるアカウント |
| 推奨 | 引用RTを好意的に受け止めてくれそうなアカウント |

### ツイート内容の条件

**引用するべきツイート:**
- ADHD・発達障害の当事者あるある
- タスク管理・先延ばし・集中力の話題
- AI活用（ChatGPT、Claude等）の話題
- 仕事術・ライフハック系で共感できる内容
- 悩みを吐露している投稿（押し付けずに寄り添える）

**引用を避けるべきツイート:**
- 議論・炎上系の話題
- 政治・宗教・センシティブな話題
- 宣伝色が強い投稿（宣伝に乗っかると嫌われる）
- ネガティブすぎる内容（関わると巻き込まれる）
- 1週間以上前の古いツイート

### 検索キーワード例

```
ADHD タスク
ADHD 先延ばし
ADHD 集中
ADHD 仕事
ADHD あるある
発達障害 仕事
ChatGPT ADHD
Claude AI 活用
```

## Part 2: どういう内容で引用ツイートするか

### 基本方針

- **宣伝しない**: 引用ツイートで記事URLを貼らない
- **共感ベース**: 上から目線のアドバイスをしない
- **価値追加**: 元ツイートに「+α」の情報を添える
- **自然体**: 押し付けがましくならない

### 引用ツイートのパターン

#### 1. 共感型（最も使いやすい）

```
これ、僕のことだ...!

[自分も同じ経験があることを一言]
```

**例:**
```
これ、僕のことだ...!

午前中「やるぞ!」って思ったタスク、気づいたら夕方になってる😇
```

#### 2. 体験談型（説得力がある）

```
わかりすぎる...!

[Before: 自分もこうだった]
↓
[After: こうしたら変わった]
```

**例:**
```
わかりすぎる...!

僕もTodoリスト何度も挫折した
↓
Claudeに「今日やる最小タスク1つだけ教えて」って聞くようにしたら続いてる👍
```

#### 3. 価値追加型（情報提供）

```
[元ツイートへの同意]

それに加えて、[追加の視点やTips]
```

**例:**
```
これマジでそう!

加えて「30分で終わるサイズに分解して」ってAIに頼むのもおすすめ
重いタスクが急にハードル下がる👍
```

### 文体ルール

**DO（やるべきこと）:**
- 「!」を使う（「〜になった!」）
- 「...」「...!」を使う（余韻を残す）
- 絵文字を1-2個使う（😇👍😭）
- 口語表現を使う（「マジで」「だいぶ」）
- 短く切る

**DON'T（避けること）:**
- 堅すぎる表現（「〜するようになりました。」）
- 上から目線（「わかりますよね？」「心当たりありませんか？」）
- 宣伝臭が強い表現
- 絵文字の使いすぎ
- 記事URLの添付

### NG表現

```
❌ 「これ読めば解決しますよ」（押し付け）
❌ 「僕の記事で詳しく書いてます」（宣伝臭い）
❌ 「〜すべきです」（上から目線）
❌ 「〜ものです」（AIっぽい）
❌ 「心当たり、ありますよね」（押し付け）
```

### OK表現

```
✅ 「これ、僕のことだ...!」（共感）
✅ 「わかりすぎる😇」（共感）
✅ 「〜したら変わった!」（体験談）
✅ 「マジでおすすめ👍」（カジュアル）
```

## Part 3: 文字数ルール

| 文字種 | カウント |
|--------|----------|
| 半角英数 | 1文字 |
| 日本語 | 2文字 |
| 絵文字 | 2文字 |
| URL | 23文字固定 |

**目安:**
- 日本語メインの引用ツイート: **140文字以内**
- 80〜100文字程度が読みやすい

---

## Part 4: クールダウンルール

- **同じアカウントへの引用RTは1週間（7日）あける**
- 引用前に `output/x_posts/quote_history.md` を確認
- 引用後は必ず履歴を更新
