# ルールについて

このフォルダには、エージェントが参照するルールとナレッジが含まれています。

---

## ファイル一覧

| ファイル | 内容 | 編集者 |
|---------|------|--------|
| `writing-rules.md` | 記事執筆のルール | 人間 |
| `review-rules.md` | レビュー時のチェックポイント | 人間 |
| `x-posting-rules.md` | X投稿のルール | 人間 |
| `strategy-rules.md` | 戦略ルール | 人間 |
| `ai-learnings.md` | AIが学んだことのログ | AI |
| `success_patterns.md` | 成功パターン集 | 人間 + AI |

---

## 使い方

### エージェントが参照する
各エージェントは、作業開始前にこれらのルールを読み込みます。

### 人間が編集する
- `writing-rules.md`: 文体や構成のルールを追加・修正
- `x-posting-rules.md`: ツイートのルールを追加・修正
- `strategy-rules.md`: 戦略方針を追加・修正

### AIが学習する
- `ai-learnings.md`: 振り返り時にAIが学んだことを追記
- `success_patterns.md`: うまくいったパターンを蓄積

---

## テーマ設定について

> **ターゲットやメインテーマは `CLAUDE.md` の「テーマ設定」セクションで管理**

CLAUDE.mdのテーマ設定を変更すると、別のトピックでも同じ仕組みで運営できます。
