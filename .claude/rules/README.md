# ルール/ナレッジベース

ADHD向けnote有料記事ビジネスの戦略を立てる際に参照するルールとナレッジを蓄積するフォルダです。

## ファイル構成

| ファイル | 用途 | 書き込む人 |
|---------|------|-----------|
| `strategy-rules.md` | 戦略・方針のルール | 人間 |
| `writing-rules.md` | 記事執筆のルール | 人間 |
| `ai-learnings.md` | 反省から学んだナレッジ | AI（/reflection実行時） |

## 使い方

### 人間が書き込むとき
- `strategy-rules.md` や `writing-rules.md` を直接編集
- 「こうしたい」「こうすべき」というルールを追記

### AIが書き込むとき
- `/reflection` スキル実行時に `ai-learnings.md` へ追記
- 記事公開後の振り返りで「うまくいったこと」「改善点」を蓄積

### スキルが参照するとき
- 各スキル（/trend-collect, /create-outline など）は実行前にこのフォルダを読む
- 蓄積されたルールに従って戦略を立てる

## ルールの優先度

1. **人間のルール** > AIの学習
2. **最新のルール** > 古いルール
3. **具体的なルール** > 抽象的なルール
